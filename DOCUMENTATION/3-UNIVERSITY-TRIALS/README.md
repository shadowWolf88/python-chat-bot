Clinician & Patient Trial Package

This folder contains materials to support a supervised pilot study using the Healing Space platform.

Included
- Clinician_Guide.md — clinician-facing instructions and safety workflows
- Patient_Guide.md — participant-facing instructions and FAQs
- Study_Protocol.md — trial design and measures
- Consent_Form.md — consent template
- Recruitment_Email.md — email template
- Presentation_Slides.md — slide deck (Markdown)
- Data_Use_and_GDPR.md — data handling and anonymization notes
- Quickstart_Local_Testing.md — local test commands and notes
- Evaluation_Forms.md — clinician/patient post-session and end-of-trial forms
- Measurement_Scales.md — PHQ-9/GAD-7 information
- FAQ.md — common questions
- Support_Contacts.md — contact list
- Ethics_Checklist.md — IRB and safety checklist

How to use
- Review `Study_Protocol.md` and `Consent_Form.md` before contacting your IRB.
- Web users (clinicians & patients): follow `Web_Access_Guide.md` — no installation required.
- Developers / admins: local quickstart to validate the app is in `Quickstart_Local_Testing.md`.
- Use `Evaluation_Forms.md` for session-level feedback collection.
