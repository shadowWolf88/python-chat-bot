## üèóÔ∏è Architecture Overview

### Technology Stack
- **Backend**: Flask (Python 3.8+)
- **Database**: PostgreSQL (production) / SQLite (dev)
- **Authentication**: JWT + Argon2/bcrypt
- **Encryption**: Fernet (AES-128)
- **LLM**: Groq (Claude/Mixtral)
- **Deployment**: Railway.app (gunicorn + Docker)

### Key Modules
- `api.py` - Main Flask application (210+ routes)
- `secrets_manager.py` - Secrets & Vault integration
- `audit.py` - Audit logging
- `training_data_manager.py` - GDPR & training data
- `fhir_export.py` - FHIR export (integrated into API)

### Database
- PostgreSQL in production
- 43 tables for users, messages, mood logs, appointments, etc.
- Auto-initialized on startup with `init_postgresql.py`
